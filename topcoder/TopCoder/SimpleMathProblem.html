Problem Statement

    

You are given four positive integers a,b,c and m. Let x = bc. Calculate and return the following value: (ax mod m).

Definition

    
Class:SimpleMathProblem
Method:calculate
Parameters:int, int, int, int
Returns:int
Method signature:int calculate(int a, int b, int c, int m)
(be sure your method is public)

Limits

    
Time limit (s):2.000
Memory limit (MB):256

Constraints

-a,b,c and m will each be between 1 and 737,373,737, inclusive.
